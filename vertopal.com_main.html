Topology

Wenda

Topological Space

Continuous mappings f : ℝn → ℝm is usually defined using metrics: ∀ϵ > 0, ∃δ > 0, s.t. d(x,x0) < δ gives d(f(x),f(x0)) < ϵ. But can we define continuity without any notion of metric?

The answer is yes. We can define continuity using only topological structures on sets.

Basic Definition

Definition 1 (Topological Space). Suppose we have a set X. A topology 𝒯 is a set of the subsets of X with the following properties:

  1. ⌀, X ∈ 𝒯.

  2. (Closed under arbitrary union) ∀i ∈ ℐ, Ui ∈ 𝒯, then ∪i ∈ ℐUi ∈ 𝒯.

  3. (Closed under finite intersection) U1, …, Un, then $\cap_{i=1}^n U_i\in \mathcal T$.

Remark. A subset U ∈ 𝒯 is also called an open set. A subset U ∈ X is called a closed set if and only if X − U is open.

Some examples.

  • 𝒯 = {⌀, X}, trivial topology.

  • Power set: 𝒯 = P(X) := {All subsets of X}, dubbed discrete topology.

Definition 2 (Metric). Let X be a set. A metric on X is a mapping X × X → ℝ ≥ 0, subject to the following.

  1. d(x,y) = 0 ⇔ x = y.

  2. d(x,y) = d(y,x), ∀x, y ∈ X.

  3. (Triangular inequality) d(x,y) + d(y,z) ≥ d(x,z), ∀x, y, z ∈ X.

Definition 3 (Open Ball). Let X be a set and d is a metric on X. We define an open ball at x0 with radius ϵ as B(x0,ϵ) := {x ∈ X | d(x,x0) < ϵ}.

Definition 4 (Metric Topology). Let X be a set and d is a metric on X. Then for any subset U ∈ X, U is open if and only if ∀x ∈ U, ∃ϵ > 0, s.t.B(x,ϵ) ⊆ U

Remark. This definition satisfies all the requirements of topological space. The proof is trivial.

Moreover, all subset U ∈ 𝒯 can be written as the union of open balls. U = ∪x ∈ UB(x,ϵ).

Definition 5 (Subspace Topology). Let (X,𝒯) be a topological space. Suppose S ⊂ X is a subset of X, then a subspace topology on S induced by 𝒯 is 𝒯′ = {U ∈ S | U = S ∩ V, V ∈ 𝒯}.

Remark. Note that an open set U ⊂ S, U ∈ 𝒯′ is not necessarily open at (X,𝒯) (i.e., U ∈ 𝒯). A simple example is S = [0,1] ⊂ ℝ and 𝒯 = {(a,b) | a, b ∈ ℝ}, where U = [0, 0.5) ∈ 𝒯′ but U ∉ 𝒯.

Continuous Mapping

Now we have defined topology on sets. It’s time to try defining continuity without using the notion of metric. Remember that in metric topology, we can define continuity as ∀ϵ > 0, ∃δ > 0, |x−x0| < δ yields |f(x)−f(x0)| < ϵ. This is in fact equivalent to f−1(f(x0) − ϵ, f(x0+ϵ)) is open in ℝ.

Definition 6 (Continuous Mapping). Let X, Y be two topological space. We call a mapping f : X → Y continuous, if ∀U ⊆ Y that is open, f−1(U) ⊆ X is open.

Remark. The composition of continuous mappings is also continuous. Also observe that definition 6 reduces to the continuous mapping in Calculus when Y is a metric topology, in which case we can show ∀x ∈ X, ϵ > 0, ∃U ∈ 𝒯X with x ∈ 𝒯X, U ⊆ B(f(x),ϵ) is equivalent to definition 6.

Definition 6 also provides us a useful trick for defining a topology in space X. Let 𝒯Xf := {f−1(U) : U ∈ 𝒯Y}, then 𝒯X(f) is a topology on f−1(Y) ⊆ X.

The definition of continuous mapping bridges across two topological space.

Definition 7 (Homeomorphism). Let X, Y be two topological space. A mapping f : X → Y is a homeomorphism, denoted X ≅ Y, if

  1. f is continuous

  2. f is bijective.

  3. f−1 is also continuous.

Remark. Homeomorphism is typically hard to disprove. It would take some effort to proof ℝ ≆ ℝ2, and even more effort to show ℝ2 ≆ ℝ3.

Some examples.

  • (Embedding) Let f : X → Y be continuous and injective. f is called an embedding from X to Y, if f : X → f(X) is homeomorphism.

    f : ℝ → ℝ2, f(x) = (ax,bx) is an embedding.

  • (Stereographic projection) S2 = {(x,y,z) ∈ ℝ3 | x2 + y2 + z2 = 1}. Then S2 ∖ {(0,0,1)} ≅ ℝ2.

    $\varphi(x,y,z) = (\frac{2x}{1-z}, \frac{2y}{1-z} -1)$ and $\varphi^{-1}(x,y-1) = (\frac{4x}{4+x^2 + y^2}, \frac{4y}{4+x^2+y^2}, \frac{-4+x^2 + y^2}{4+x^2 + y^2})$

Next, we build a useful tool for proving continuous mappings. The following lemma allows us to check the continuity of a mapping by checking continuity in its segments.

Lemma 8 (Pasting Lemma). Let X, Y be two topological spaces, {Xi | i ∈ I} be a family of subspaces of X. Let {fi | i ∈ I} be a family of continuous functions and fi|Xi ∩ Xj=fj|Xi ∩ Xj for ∀i, j ∈ I. If

  • Xi are open sets in X.

  • Or, Xi are closed and I is finite.

Then f : X → Y with f|Xi = fi is continuous.

Products of Topology

Let X and Y be two topological space. We want to find topology on X × Y. However, U × V, U ∈ 𝒯X, V ∈ 𝒯Y does not naturally form a topology, as it is not closed under union.

Definition 9 (Product Topology). Let (X,𝒯X), (Y,𝒯Y) be two topological space. We define open boxes in X × Y as U × V, where U ∈ 𝒯X and V ∈ 𝒯Y.

Moreover, we call a subset W ⊆ X × Y open, if it is a union of open boxes.

Remark. To check this is indeed a topology is trivial. The most crucial property is that intersecting two open boxes (U1×V1) ∩ (U2×V2) = (U1∩U2) × (V1∩V2) is still an open box.

Basis

We have seen in previous examples that a topology space may be constructed from some basic ingredients, e.g., open balls or open boxes. In this section, we formalize this intuition by characterising open balls and open boxed as basis.

Definition 10 (Basis for a topology). Let X be a set. ℬ be a family of subsets of X. We call ℬ a basis for a topology if

  • (Covers X) ∀x ∈ ℬ, ∃U ∈ ℬ, such that x ∈ U.

  • (Intersection) If U, V ∈ ℬ, there exists a collection of subsets {Wi ∈ ℬ | i ∈ I} such that U ∩ V = ∪i ∈ IWi.

Definition 11 (Subbasis). Let X be a set. S ⊆ 2X is a subbasis if ⋃Si ∈ SSi = X.  (only need S cover X)

Given a subbasis, how do we generate a basis? Subbasis S → Basis ℬ → Topology 𝒯

Let ℬS = {Si1 ∩ … ∩ Sim | Si1, …, Sim ∈ S}. Then ℬS is a basis.

Let 𝒯ℬ = {⋃i ∈ ℐBi | Bi ∈ ℬ}. Then 𝒯ℬ is a topology.

Examples.

  • Metric topology (X,d), corresponding to basis open balls {B(x,ϵ) | x ∈ X, ϵ > 0}.

  • Product topology X × Y, corresponding to basis open boxes {U × V | U ∈ X, V ∈ Y}.

  • S = {(−∞,a) | a∈ℝ}∪{(b,∞) | b ∈ ℝ} is a subbasis in ℝ. Its corresponding basis is ℬS = {(a,b) | ∀a, b ∈ ℝ} which is already a topology in ℝ.

  • (Initial topology) Consider some continuous mappings fi : X → Yi. Let S = {fi−1(U) | U ⊂ Yi open}. Then 𝒯S is called an initial topology.

Proposition 12. f : X → Y, S is a subbasis of the topology of Y. Then f is continuous ⇔ f−1(Si) ∈ 𝒯X, ∀Si ∈ S

Hausdorff Space

Definition 13 (Hausdorff space). A topological space X is called Hausdorff if for any pair of points x, y ∈ X, there exists an open neighborhood U of x and an open neighborhood V of y such that U ∩ V = ⌀.

Quotient Topology and Gluing

Basic Definition

Relation. A relation in X is a subset R ⊂ X × X. (x,y) ∈ R is denoted x ∼ y.

An equivalent relation satisfies the following properties:

  • (reflexive) x ∼ x

  • (symmetric) x ∼ y ⇒ y ∼ x

  • (transitive) x ∼ y, y ∼ z ⇒ x ∼ z

Consider two mappings p1, p2 : T → X. We can define p : T → X × X, such that p(t) = (p1(t),p2(t)). Then Rp1p2 = p(T) is a relation generated by p1 and p2.

Definition 14 (Equivalent class). Let ∼ be an equivalent relation. An equivalent class containing x is denoted by  := {y ∈ X | x ∼ y}.

Definition 15 (Quotient sets). The quotient set in X of an equivalent relation ∼ is denoted by X/∼  := {[x] | x ∈ X}.

Quotient Topology

Definition 16 (Quotient Topology). (X,𝒯X) is a topology space. Let f : X → Y be a surjective quotient. U ⊂ Y is open if f−1(U) is open. The quotient topology is defined as 𝒯Y = {U ⊂ Y | f−1(U) ∈ 𝒯X}

Remark.

  • 𝒯Y is the finest topology that makes f continuous.

  • For an equivalent relation ∼, consider its projection map π : X → X/∼. The quotient topology is 𝒯 = {U ⊂ X/∼ | π−1(U) ∈ 𝒯X}

Definition 17 (Quotient map). Let X and Y be topological spaces. Suppose f : X → Y is a continuous and surjective. If the quotient topology is homeomorphic to the topology on Y, i.e., (X/∼f,QT) ≅ (Y,𝒯Y), then f is called a quotient map.

The question is: how do we decide a map is a quotient map or not? The following propositions 18 and 19 are usually used jointly to prove a quotient map.

Proposition 18. Let f : X → Y be continuous and surjective. Let {Ui}i ∈ ℐ be an open cover of Y. We denote the restriction of f on Ui by fi : f−1(Ui) → Ui with fi = f|f−1(Ui).

Proposition 19. Let f : X → Y be continuous and surjective. If f has a section S : Y → X, i.e., S : Y → X continuous and f ∘ S = IdY, then f is a quotient map.

Remark. The existence of a section S implies f is surjective.

Proposition 20. Let f : X → Y be a continuous, surjective map and f is open (or closed) map. Then f is a quotient map.

Gluing Open Subsets

Definition 21 (Open gluing datum). An open gluing datum consists of the following:

  • (What to glue) A family of topological spaces {Ui}i ∈ ℐ.

  • (Where to glue) Open subspaces Uij ⊂ Ui, we also set Uii = Ui.

  • (How to glue) homeomorphism fij : Uij → Uji

satisfying the following

  • fii = IdUi

  • fij(Uij∩Uik) ⊂ Uji ∩ Ujk

  • (Cocycle condition) The homeomorphisms agree on triple intersections fik|Uij ∩ Uik=fjk|Uji ∩ Ujk ∘ fij|Uij ∩ Uik

Remark. The cocycle condition guarantees the consistency when gluing  > 2 subsets together.

Definition 22 (Disjoint union). The disjoint union of a collection of sets {Ui}i ∈ ℐ: ⨆i ∈ ℐUi = {(x,i) | i ∈ ℐ, x ∈ Ui}

Definition 23 (Open gluing: (Ui,Uij,fij)). U = ⨆i ∈ ℐUi.  ∼  : (x,i) ∼ (y,i) ⇔ x ∈ Uij, y ∈ Uji, fij(x) = y. Check that this is indeed an equivalence relation by definition. We call (U/∼,QT) the topological space glued from datum (Ui,Uij,fij).

Manifold

Definition 24 (Chart). Let X be a topological space. An n-dimensional chart on X consists of an open subset U ⊂ X equipped with the subspace topology and a homeomorphism ϕU → V, where V is an open subspace of ℝn with the standard topology.

Definition 25. Let M be a Hausdorff space with a countable topological basis (also called second countable). M is a topological manifold (of pure dimension n) if there exists a collection of (n- dimensional) charts (Ui,φi)i ∈ I such that M = ⋃i ∈ IUi.

Connectedness

Connected Spaces

Definition 26 (Connected Space). A topological space X is connected if it is not the union of two disjoint non-empty open subsets.

Remark.

X is connected. ⇔ The subsets that are open and closed in X must be ⌀ and X.

Connectedness and continuous maps

Proposition 27. Let f : X → Y be a continuous map of topological spaces. Assume X is connected, then f(X) is connected.

In order words, connectedness is a topological invariant.

Using this property, we can easily show ℝ ≆ ℝ2. Since f(ℝ∖{0}) is a connected subspace of ℝ2.

Connectedness and subspaces

Proposition 28. Let X be a topological space and X′ ⊆ X is a connected dense subset, then X is also connected.

Remark. A subset A ⊆ X is dense, if its closed envelop $\overline A = X$, where $\overline A = \bigcap_{i\in I} V_i$, Vi is closed and A ⊆ Vi. An example: $\overline {\mathbb Q} = \mathbb R$.

Proof. Suppose X is not connected, then there exists a nonempty proper subset U ⊂ X that is closed and open. Since X′ is dense, U ∩ X′ ≠ ⌀. Then U ∩ X′ ⊆ X′ is closed and open in X′, which means U ∩ X′ = X′. Thus, $U = X^\prime = \overline X^\prime = X$, contradict. ◻

Corollary 1. If A ⊆ X and A is connected, then its closed envelop $\overline A$ is also connected

Locally constant function

Definition 29 (Locally constant function). Let Λ be a topological space with discrete topology. A continuous map f : X → Λ is locally constant if ∀x, ∃U ∈ 𝒯, x ∈ U, such that f|U is constant.

Proposition 30. Let (Λ,discrete) be a topological space with at least two elements. X is a topological space. Then X is connected if and only if any locally constant function f : X → Λ is constant.

Using proposition 30, we can prove the following convenient proposition for connectedness.

Proposition 31. Let {Ai}i ∈ I be a collection of connected subspaces of X. Let B be a connected subspace such that B ∩ Ai ≠ ⌀, then B ∪ ⋃i ∈ IAi is connected.

Proof. Define A = B ∪ ⋃i ∈ IAi. Let f : A → Λ be a locally constant function. We know f|Ai and fB are all constant function as Ai and B are connected. Thus, f|A=f|U is constant. ◻

Corollary 2. (X,𝒯), if ∀x, y ∈ X, there exists a connected subspace U, x, y ∈ U, then X is connected.

Corollary 3. X, Y are connected ⇒ X × Y is connected.

Corollary 4. f : X → Y is a surjective continuous map. Y has the quotient topology. Then X is connected ⇒ Y is connected.

Remark. Based on these corollaries above, we can show some basic spaces are connected.

  • ℝn is connected. (from corollary 3)

  • Sn is connected. Proof: for x, y ∈ Sn, choose a z ∈ Sn and z ≠ x, z ≠ y. Then ℝn/{z} ≅ ℝn is connected.

  • ℝPn and ℂPn connected from corollary 4.

Path Connectedness

Definition 32 (Path). Let X be a topological space. A path γ is a continuous map γ : I → X. γ(0) and γ(1) are the start and the end of the path.

Definition 33 (Path connected). A topological space X is path connected if for any x, y ∈ X, ∃ a path γ with γ(0) = x, γ(1) = y.

Connected Components

Definition 34 (connected components). (X,𝒯). x, y are mutually connected if there is a connected subspace U ⊂ X and x, y ∈ U.

Mutual connectedness is an equivalence relation. Its equivalence class are the connected components of X.

Total number of connected components is a topological invariant: π0(X).

Without any difficulty we can extend the mutual connectedness to mutual path connectedness. Note that mutual path connectedness is also an equivalence relation. Similarly we define path connected components.

Compactness

Compactness seeks to generalize the property of a bounded and closed subset of Euclidean space.

Definition 35 (Compactness). Let X, 𝒯 be a topological space. C is compact if ∀ open cover {ui}i ∈ ℐ of X, there exists Ui1, …, Uin ∈ {Ui}i ∈ ℐ, such that $X\subset \bigcup_{i=1}^n U_{i_j}$.

Remark. In other words, any open cover of X contains a finite open cover of X.

Examples.

  • (X,trivial) is compact. If |X| < ∞, (X,discrete) is compact.

  • [0,1] is compact.

  • (0,1) is not compact, because $\{(\frac{1}{n},1)\}_{n=1}^\infty$ is an open cover, but no finite subset of it is a cover.

Basic Property of Compact Spaces

Proposition 36. If f is continuous, X is compact, then f(X) is also compact.

With subspace

Proposition 37. X is a compact, closed space. Z is a closed subspace of X. Then Z is also compact.

Remark. This proposition 37 guarantees that every bounded closed subset of ℝ is compact.

Proposition 38. X is Hausdorff. Z ⊂ X is compact ⇒ Z is closed.

With product space

Definition 39 (Refinement of an open cover). 𝒰 = {Ui}i ∈ I, 𝒱 = {Vi}i ∈ I are open covers of X. If ∀Vi, ∃Uk, such that Vi ⊂ Uk, then 𝒱 is called a refinement of 𝒰.

Proposition 40. If X, Y are compact topological spaces, then X × Y is also compact.

Proof Sketch.. Let 𝒰 be an open cover of X × Y, we can find a refinement of 𝒱 of 𝒰, where 𝒱 only contains the open boxes in X × Y. We can easily find a finite open cover in 𝒱 using compactness of X and Y. ◻

Theorem 41 (Heine-Borel). Let Z be a subspace of ℝn. Z is compact if and only if Z is a bounded and closed subset.

Proof. If Z is compact, then since ℝn is Hausdorff, we know Z is closed according to proposition 38. Note that Z ⊂ ⋃r > 0B(0,r), so Z ⊂ ⋃r = r1, …, rnB(0,r), which means Z is bounded.

The reverse direction is direct since Z is a closed subspace of [−N,N]n for sufficiently large N. ◻

Compact open topology

In this section, we consider the set of all continuous map from X to Y, dubbed 𝒞[X;Y], and assign a topology on it using compactness.

Definition 42 (Compact open topology). Consider the set of all continuous map from X to Y: 𝒞[X;Y]. Define a subbasis = {ℳK, U | K ⊂ X compact, U ⊂ Y open}, where ℳK, U = {f ∈ 𝒞[X;Y] | f(K) ⊂ U}. The topology generated by this subbasis is called a compact open topology.

Moreover, if (Y,d) is a metric topological space, then we have

Proposition 43. (Y,d) metric space. A collection of sets BK(f,ϵ) = {g ∈ 𝒞[X;Y] | supx ∈ Kd(f(x),g(x)) < ϵ}. with K ⊂ X compact. Then {BK(f,ϵ)} is a basis of compact open topology.

Example.

  • The solution space for $\ddot x + x = 0$: ℳ = {x = acos t + bsin t | (a,b) ∈ ℝ2} is homeomorphic to ℝ2.

Proposition 44. Y can be embedded into 𝒞[X;Y]. Consider a bijective map c : Y → 𝒞[X;Y] such that y ↦ f(X) = y.

Proposition 45. 𝒞[X;Y] is Hausdorff if and only if Y is Hausdorff.

Proposition 46. 𝒞[X;ℝ] is path connected.

Proof. f : X → ℝ. Let cf : [0,1] → 𝒞[X;ℝ] : cf(t) = t ⋅ f. ◻

Homotopy

Homeomorphism is good for sure, but usually too restricted to be held. Here we discuss a weaker condition, i.e., Homotopy.

Homotopy of maps

Definition 47. Let f, g : X → Y be two continuous maps. A homotopy between f and g is a continuous map σ : X × [0,1] → Y such that ∀x ∈ X, σ(x,0) = f(x), σ(x,1) = g(x). f and g is then homotopic, denoted f ≃ g.

Remark.

  • σ defines a path in 𝒞[X;Y].

  • Be cautious that σ must be continuous on X × [0,1]. Only continuous for σ(⋅,t) is not enough.

  • If f is homotopic to a constant function, it is called null-homotopic.

Proposition 48. Homotopy f ≃ g is an equivalence relation.

Remark. We denote the quotient set 𝒞[X;Y]/≃ as [X;Y].

Proposition 49. Let f1, f2 : X → Y and g1, g2 : Y → Z. If f1 ≃ f2 and g1 ≃ g2, then g1 ∘ f1 ≃ g2 ∘ f2.

Proof Sketch.. We prove by g1 ∘ f2 ≃ g1 ∘ f1 ≃ g2 ∘ f2.

The first relation can be proved by g1 ∘ σ1. The second can be shown by σ2′(x,t) = σ2(f2(x),t). ◻

Homotopy equivalence.

We consider the homotopic inverse. For f : X → Y we require its homotopic inverse to satisfy: f ∘ g ≃ IdY and g ∘ f ≃ IdX.

Definition 50 (Homotopy equivalence). Topological spaces X and Y are called homotopy equivalent if there exist continuous maps f : X → Y and g : Y → X such that f ∘ g ≃ IdY, g ∘ f ≃ IdX.

Examples.

  • ℝn ≃ {x}.

Definition 51 (Contractible space). X is a contractible space if X ≃ {x}. This is equivalent to the condition that IdX is null-homotopic.

Example.

  • ℝ ∖ {0} ≃ {p1} ∪ {p2}.

  • ℝ2 ∖ {0} ≃ S1.

To better describe the above observation, we define defomation retraction.

Definition 52 (Deformation retraction). Let X be a topological space and A be a subspace of X. A deformation retraction is a homotopy r : X × [0,1] → X such that r(x,0) = x and r(x,1) ∈ A. Moreover, for any a ∈ A and t ∈ [0,1] we have r(a,t) = a.

Fundamental Group

Definition 53 (Loop). Let X be a topological space and γ : [0,1] → X is a path on X. If γ(0) = γ(1) we call it a loop.

Definition 54 (Paths). The inverse of a path γ is γ−(t) = γ(1−t). The constant path I[x] is defined as γ(t) = x for some x ∈ X.

Definition 55 (Path homotopy). Two paths with same initial and end points: γ1, γ2 : [0,1] → X. γ1≃pγ2 if there exists a continuous map σ : [0,1] × [0,1] → X with σ(t,0) = γ1(t) and σ(t,1) = γ2(t). Moreover, we need σ(0,s) = γ1(0) = γ2(0) and σ(1,s) = γ1(1) = γ2(1).

Remark. Path homotopy is stricter than map homotopy as it fixes the initial and end points during deformation. To see their difference, consider ℝ ∖ {0}.

Definition 56 (Path multiplication). Let α, β be paths on a topological space X. Suppose α(1) = β(0). We define $$(\alpha *\beta)(s) = \begin{cases} \alpha(2s),\ s\in [0,\frac{1}{2}],\\ \beta(2s-1),\ s\in [\frac{1}{2},1]. \end{cases}$$

Proposition 57. For a path γ on X we have γ * γ−≃pI[γ(0)]

Note that α≃pα′ and β≃pβ′ yields α * β≃pα′ * β′. We can define p * [β]p = [α*β]p

Proposition 58. Suppose α, β, γ are paths on X. Then p * ([β]p*[γ]p) = ([α]p*[β]p) * [γ]p.

We now want to use define a group structure on this operation. However, not all paths on X can be multiplied as they might not have overlapped end points. To fix this issue, we consider loops with a fixed end points.

Definition 59 (Fundamental Group). Let X be a topological space. Let x ∈ X. We define π1(X,x) = {[γ]p | γ loops in X, i.e., γ(0) = γ(1) = x}. Then (π1(X,x),*) is a group.

Remark.

  • On ℝ2 all loops are path homotopic to the constant path. On ℝ2 ∖ {0}, however, loops that wrap around 0 are not path homotopic to the constant path.

  • A fundamental group is not necessarily Abelian.

Proposition 60. π1((X,x),*) is a homotopic invariant.

Examples.

  • Let A be a band and B be a Mobius band. Then π1(A,*) ≅ π1(B,*) ≅ π1(S1,*) ≅ (ℤ,+).

Proposition 61. If X is path connected, then π1((X,x),*) ≅ π1((X,y),*) for any x, y ∈ X.

Remark. Let X be a topological space, then the followings are equivalent

  • π1(X,x) is trivial

  • ∀γ1, γ2 paths with γ1(0) = γ2(0) = x and γ1(1) = γ2(1), we have γ1≃pγ2.

  • All loops at x ≃p constant loop at x.

Covering Space

Definition 62 (Covering Space). Let (E,p) be a space over p with p : E → B continuous surjective. p is a covering map and (E,p) is a covering space over B, if ∀b ∈ B there exists an open neighbor U of b such that p−1(U) = ⋃i ∈ IUi where Ui are disjoint open subsets of E and p|Ui : Ui → U is a homeomorphism.

Lemma 63 (lifting lemma). Let p : E → B be a covering map. Let c : [0,1] → B be a path on B. If b̃ ∈ E, p(b̃) = c(0) = b, then there exists a unique path in E: c̃ : [0,1] → E, such that c̃(0) = b̃ and p ∘ c̃ = c.

Proposition 64. Let p : E → B be a covering map. Let c1, c2 : [0,1] → B be two homotopic paths on B. Suppose c1(0) = c2(0) = b. Then c̃1, c̃2 with c̃1(0) = c̃2(0) = b̃ ∈ p−1(b) are also homotopic.

Higher Homotopic Group

In discussion of path-connected components, we note that π0(X) = {[x→X] | x ∈ X} can be viewed as the set of all homotopic equivalent classes from a single point to X. Similarly, π1(X) = {[[0,1]→X] | x ∈ X} can be viewed as the set of all homotopic equivalent classes of loops in X. We want to generalize 0-dim point and 1-dim loops to n dimension in this section.

Definition 65 (n-cube). In = [0,1]⊗n = {(t1,…,tn) ∈ ℝn | ti ∈ [0,1]}. Then ∂In = {(t1,…,tn) ∈ In | ∃ti = 0, 1}.

Moreover, we define a n-loop based at x ∈ X be a continuous map γ : In → X, s.t., γ|∂In = x.

Remark. This requirement is a generalization of the requirement γ(0) = γ(1) in the 1-dimensional case, i.e., loops.

Definition 66 (Homotopy for n-loops). Let γ, σ be two n-loops based at x. Let γ(x) = σ(x) for any x ∈ ∂In. Then γ ≃ σ homotopic as n-loops if there exists H : In × I → X such that H(t;0) = γ(t), H(t;1) = σ(t), and H(t;s) = x, ∀t ∈ ∂In, s ∈ I.

To construct group structures for n-loops, we define their composition as $$\sigma_1 *\sigma_2 = \begin{cases} \sigma_1(2t_1,t_2,\dots, t_n),\quad 0 \leq t_1\leq 1/2.\\ \sigma_2(2t_1 - 1, t_2, \dots, t_n), \frac{1}{2}\leq t_1\leq 1. \end{cases} \label{nloop}$$

Remark.

  • Surprisingly, σ1 * σ2 ≃ σ2 * σ1 for n ≥ 2. In other words, (πn(X,x),*) is an Abelian group.

  • Moreover, [σ1*σ2]p well defined, in the sense that choosing the first index in [nloop] is equivalent to choosing any other coordinate in [n].

Examples.

  • πn(Sn) ≅ ℤ, n ≥ 1.

  • π5(S2) ≅ ℤ2, π6(S2) ≅ ℤ12.

  • π1(ℝP2) ≅ ℤ2, πn(ℝPn) ≅ ℤ, n ≥ 2.

  • (Bott periodicity of Lie groups) For n ≥ (k+1)/2, $$\pi_k(U(n)) \cong \pi_k(SU(n)) \cong \begin{cases} \{e\},\ k\ \text{even}\\ \mathbb Z,\ k\ \text{odd}. \end{cases}$$

Application of Homotopy: Taxonomy of Defects

For a given material, describe its magnetic feature by its magnetic moment m⃗(x) = ⟨∑imi(x)⟩.

The free energy ℱ = ℱgrad + ℱpotential + ℱext. where ℱgrad = κ|∇m|2 and $$\mathcal F_{potential} = -\alpha(T)|m|^2 + \beta(T) |m|^4 + \frac{\alpha(T)^2}{4\beta(T)}.$$ Note that we should minimize F = ∫ℱ(x)dx. Suppose the gradient is zero, i.e., the magnetic moment is constant (homogeneous). If α(T) < 0, β(T) > 0, then |m⃗| = 0. If α(T) > 0, β(T) > 0, then |m⃗| > 0, i.e., Ferromagnetism.

We define defects as m⃗(x), F < ∞ where ℱ is a local minimum. If m⃗(x) is regular everywhere, we call it a regular defect. If m⃗(x) has a singular point, we call it a singular defect.

For regular defects, we know when |x| → ∞, ℱ → 0 or otherwise the integration would not be finite. Thus, $$\lim_{|x|\to \infty} |\vec m(x)| = m_0 \propto \sqrt{\frac{\alpha}{\beta}}.$$ The boundary condition for m⃗(x) : ℝd → ℝ3 is bounded to 𝕊2. The solution is locally stable guaranteed by homotopy on πd − 1(𝕊2).

π0(S2) = π1(S2) = {0}, π2(S2) = ℤ.

Homology

Simplex

  • 0-simplex: a point ⟨p0⟩.

  • 1-simplex: an edge: ⟨p0, p1⟩.

  • 2-simplex: an triangle ⟨p0, p1, p2⟩, but p0, p1, p2 do not lie on a same line.

Geometric independence: r + 1 points are geometrically independent, if p0, …, pr ⊆ r − dim hyperplane, but p0, …, pr ⊈ (r−1) − dim hyperplane.

Definition 67 (r-simplex). {p0, …, pr} be (r+1)-geometrically independent. p0, …, pr ∈ ℝm, m ≥ r. A r-simplex σr(⟨p0,…,pr⟩) is defined as $$\sigma^r = \{x\in \mathbb R^m \ |\ x = \sum_{i=1}^r c_i p_i, c_i \geq 0, \sum_{i=0}^r c_i = 1\}$$

Remark. {ci} are called the barycentric coordinates of x.

Definition 68 (q-face). Let q ∈ ℤ, 0 ≤ q ≤ r. Select q + 1 points from {p0, …, pr} and denote them by {pi0, …, piq}. Then the q-simplex ⟨pi0, …, piq⟩ is a q-face of ⟨p0, …, pr⟩, denoted ⟨pi0, …, piq⟩ ≤ ⟨p0, …, pr⟩

Simplicial Complex

Definition 69. K is a set of simplexes. K is a simplicial complex if

  • σ ∈ K, σ′ ≤ σ ⇒ σ′ ∈ K

  • For any σ, σ′ ∈ K, either σ ∩ σ′ = ⌀, or σ ∩ σ′ ≤ σ, σ ∩ σ′ ≤ σ′.

Remark. For a simplicial complex K, the union of all elements in K is called a polyhedron, |K| := ⋃σ ∈ Kσ.

Definition 70 (Triangulation). Let X be a topological space. If there exists a simplicial complex K, such that f : |K| → X is a homeomorphism, then X is triangulable, and (K,f) is a triangulation of X.

Chain Group

Define orientation on ⟨p1, …, pn⟩. Two simplexes have the same orientation if they differ by an even permutation.

Definition 71 (Chain group). K simplicial complex. The r-chain group on K, Cr(K), is an Abelian group generated by the oriented r-simplexes of K. If r > dim K, Cr(K) = 0. An element in Cr(K) is called an r-chain.

Remark. Cr(K) = {∑iziσir | zi ∈ ℤ, σir ∈ K}.

  • For c = ∑iciσir, d = ∑idiσir, c + d = ∑i(ci+di)σir.

  • Cr(K) ≅ ℤd for some d. d is the dimension of Cr(K) and d= the number of r-simplex in K.

  • Consider dim K = 2, dim C2(K) − dim C1(K) + dim0(K) is the Euler characteristic for planar graphs, which is invariant under homotopy.

Next, we define cycle groups and boundary groups that are subgroups of chain groups. We start with defining what a boundary is.

Definition 72 (Boundary). Note that the boundary of a r-simplex is a union of (r−1)-simplexes it contains. We define a boundary operator ∂r : Cr(K) → Cr − 1(K): ∂r(∑iziσir) = ∑izi∂r(σir) and for σir = (p0,p1,…,pr), $$\partial_r(\sigma_i^r) = \sum_{i=0}^r (-1)^i (p_0,\dots, p_{i-1}, p_{i+1},\dots, p_r).$$ By definition we know σr is a group homomorphism.

Then the kernel ker(∂r) is called the cycle group and the image Im(∂r) is called the boundary group.

Definition 73 (Cycle group). An r-cycle group Zr(K) is defined as Zr(K) = Ker(∂r) = {c ∈ Cr(K) | ∂rc = 0}.

Definition 74 (Boundary group). An r-boundary group Br(K) is defined as Br(K) = Im∂r + 1 = {c ∈ Cr(K) | c = ∂r + 1d, d ∈ Cr + 1(K)}.

Proposition 75. ∂r ∘ ∂r + 1 : Cr + 1(K) → Cr − 1(K) is a zero map.

Therefore, Br(K) ⊂ Zr(K) ⊂ Cr(K).

Simplicial Homology

We define a chain complex (C⋅,d⋅) of Abelian groups (Cn)n ∈ ℤ and homomorphisms of Abelian groups dn : Cn + 1 → Cn subject to dn ∘ dn + 1 = 0 for all n ∈ ℤ.

Definition 76 (Homology). An r-homology group of (C⋅,d) is defined as $$H_r(C_{\cdot}, d) = \frac{Ker(d_r: C_r \to C_{r-1})}{Im(d_{r+1}: C_{r+1} \to C_r)}$$

Theorem 77. Let X,Y be two triangulable homeomorphic topological space, and (K,f), (L,g) be triangulations of X and Y. Then Hr(K) ≅ Hr(L)

Proposition 78. Let K be a connected simplicial complex, then H0(K) ≅ ℤ

Structure of Homology Group

Note that Hr(K) is Abelian. The most general form of an Abelian group is Hr(K) ≅ ℤn × ℤk1 × … × ℤkp The first part is called the free part and the second is called torsion part.

If we change the coefficient in chain groups from integer to real numbers, the torsion part disappears. So Hr(K;ℝ) ≅ ℝf.

Definition 79 (Betti number). Let K be a simplicial complex. The r-th Betti number is br(K) = dim Hr(K;ℝ) which is also the rank of the free Abelian part of Hr(K;ℤ).

Definition 80 (Euler characteristics). Let K be an n-dimensional simplicial complex, and Ir be the number of r-simplexes in K. The Euler characteristics χ is $$\chi(K) = \sum_{r=0}^n (-1)^r I_r.$$

Theorem 81 (Euler-Poincare). Let K be an n-dim simplicial complex, $$\chi(K) = \sum_{r=0}^n (-1)^r b_r(K).$$

Smith Normal Form

Let A = (aij) ∈ Mm × n(ℤ). A is in Smith normal form if

  • aij = 0, ∀i ≠ j

  • 0 ≤ r ≤ min (m,n), aii ≠ 0 for i ≤ r and aii = 0 otherwise.

  • ai = aii, 0 ≤ i ≤ r, then ai|ai + 1.

Proposition 82. If A ∈ Mm × n(ℤ), then there exists U ∈ Mm × m(ℤ), V ∈ Mn × n(ℤ), det U =  ± 1, det V =  ± 1, and D ∈ Mm × n(ℤ) is in Smith normal form. We have A = UDV.

Proposition 83. Cr − 1(K) ← Cr(K) ← Cr + 1(K) ← …. Let A be the matrix representation for ∂r + 1 and B for ∂r.

Suppose SNF(A) = diag(a1,…,ar,0), then Hr(K;ℤ) ≅ KerB/ImA ≅ ℤdim Cr(K) − r − s × ℤa1 × … × ℤar where r = rankA, s = rankB.
